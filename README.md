# PowerBI-Sales-Analytics-Dashboard
# 📊 Retail Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Retail Sales Analysis Dashboard** built using Power BI. The goal is to analyze sales performance, profitability, and product trends using a structured and user-friendly dashboard.

The report enables users to explore key business insights across different dimensions such as category, region, and time.

---

## 🎯 Objectives

* Analyze overall **sales and profit performance**
* Identify **top-performing and loss-making products**
* Understand **category and sub-category trends**
* Enable **interactive filtering and navigation**

---

## 🛠️ Tools & Technologies Used

* **Power BI Desktop**
* **Power Query (Data Cleaning & Transformation)**
* **DAX (Data Analysis Expressions)**
* **Excel Dataset (Superstore)**

---

## 📊 Dashboard Features

### 🔹 1. Executive Overview

* KPIs: Total Sales, Profit, Orders, Profit Margin
* Sales by Region and Segment
* Monthly Sales Trend
* Category-wise performance

---

### 🔹 2. Product Performance Analysis

* Matrix showing Category & Sub-Category with Sales, Profit, and Profit Margin
* **Top 10 / Bottom 5 Sub-Categories toggle** using bookmarks
* Visual comparison of product profitability
* Interactive filtering for deeper insights

---

### 🔹 3. Regional Analysis

* Sales distribution by state (Map visualization)
* Profit analysis by region
* Orders by shipping mode

---

## 🔘 Interactivity & Advanced Features

* **Synced slicers across pages** for consistent filtering
* **Navigation buttons** for smooth page transitions
* **Bookmark-based toggle** (Top/Bottom performers)
* **Reset filters button**
* Clean and user-friendly UI design

---

## 📈 Key Insights

* Certain sub-categories generate high sales but low profit
* Technology category tends to be the most profitable
* Some products consistently result in losses
* Regional performance varies significantly across states
